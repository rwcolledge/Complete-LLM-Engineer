
\chapter{Operating Systems}
\label{chap:operating-systems}

\chapterintro{An operating system manages hardware and provides services that programs depend on.}
\learningobjectives{\begin{itemize}\item Define an operating system.\item Explain processes and files.\item Understand why Linux is important for AI engineering.\end{itemize}}

\section{What Is an Operating System?}
An operating system is system software that manages hardware resources and provides common services for applications.

\section{Processes}
A process is a running program. When Python runs a script, the operating system creates a process for it.

\section{Files}
Operating systems organize persistent data into files and directories.

\section{Why Linux Matters}
Many servers, containers, Databricks runtimes, and cloud AI systems use Linux. Learning Linux commands helps LLM engineers inspect files, run programs, automate workflows, and debug production systems.

\section{Command Examples}
\begin{lstlisting}[language=bash]
pwd
ls
mkdir experiments
cd experiments
\end{lstlisting}

\section{Exercise}
\begin{exercise}What is the difference between a program and a process?\end{exercise}
\begin{solution}A program is stored instructions. A process is an executing instance of a program.\end{solution}

\section{Summary}
Operating systems bridge applications and hardware. Linux skills are foundational for practical AI work.
